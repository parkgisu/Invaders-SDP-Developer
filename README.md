# Space Invaders Game Project

## Overview

본 프로젝트는 Java 기반으로 Space Invaders 게임을 구현한 팀 프로젝트입니다.
소프트웨어 공학 수업의 일환으로 진행되었으며,
게임 시스템을 모듈 단위로 분리하여 협업 개발을 수행하였습니다.

---

## My Contribution

본 프로젝트에서 저는 **Item System 개발을 담당**하였습니다.

### 주요 작업

* 아이템 구매 및 적용 시스템 구현
* 구매 단계 제한 로직 구현 (이전 단계 아이템 선구매 필요)
* 게임 종료 시 아이템 및 코인 상태 초기화 처리
* 게임 상태 변화에 따른 아이템 효과 적용 로직 구현
* 기존 게임 로직과의 통합 및 버그 수정

---

## Key Features

* 다양한 아이템을 활용한 게임 플레이 확장
* 플레이어 상태에 따른 동적 시스템 적용
* 게임 진행에 따른 상태 관리 로직

## Team Project Context
